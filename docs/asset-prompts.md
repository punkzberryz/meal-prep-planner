# Asset Prompts

Use these prompts to generate visual assets for the meal prep planner app (ChatGPT -> Canva or similar). Each prompt includes where the asset should be used so it can be placed in the app later.

## 1) App Shell Hero Illustration
**Placement:** `/app` dashboard landing (top of the page, above the week grid).
**Asset location:** `public/assets/illustrations/app-hero.webp`
**Purpose:** Welcome moment that reinforces calm meal planning.
**Prompt:**
Create a friendly, modern flat illustration for a meal prep planner dashboard hero. Show a cozy kitchen counter with a salad bowl, cutting board, and a tidy weekly planner sheet or calendar card. Keep the scene simple with 3–4 objects max, soft shadows, and lots of negative space. Color palette: warm and bright (tangerine, sunny yellow, mint, cream). No text, no numbers, no photo realism. Output as a wide image, 1600x900.

## 2) Empty State — No Meals Yet
**Placement:** `/app/meals` empty state card.
**Asset location:** `public/assets/illustrations/empty-meals.webp`
**Purpose:** Encourage users to add their first meal.
**Prompt:**
Create a minimal flat illustration for an empty state in a meal planner app. Show a clean, empty plate or bowl with a fork and spoon resting parallel above the rim. Add a small sparkle or leaf accent to suggest freshness. No text, no numbers, no photo realism. Simple shapes, soft shadows, generous negative space. Use bright, friendly colors (tangerine, mint, cream). Output as 800x800.

## 3) Empty State — No Plan This Week
**Placement:** `/app/planner` weekly planner empty state panel.
**Asset location:** `public/assets/illustrations/empty-week.webp`
**Purpose:** Suggest starting a plan.
**Prompt:**
Create a minimal flat illustration for a weekly planner empty state. Depict a simple calendar card or week grid with a single checkmark and a bowl icon nearby. Keep shapes geometric and clean. No text, no numbers, no photo realism. Bright, playful palette (sunny yellow, coral, mint). Soft shadows, lots of negative space. Output 800x800.

## 4) Empty State — Grocery List
**Placement:** `/app/grocery` empty state panel.
**Asset location:** `public/assets/illustrations/empty-grocery.webp`
**Purpose:** Prompt users to generate a list.
**Prompt:**
Create a minimal flat illustration for an empty grocery list. Show a small grocery bag with a few simple shapes peeking out (leaf, tomato, carrot). Keep it clean, playful, and minimal. No text, no numbers, no photo realism. Bright, friendly palette and soft shadows. Output 800x800.

## 5) Loading Skeleton Placeholder Accent
**Placement:** Loading state background accents for `/app` and `/app/planner` skeletons.
**Asset location:** `public/assets/illustrations/loading-accent.webp`
**Purpose:** Subtle visual texture while content loads.
**Prompt:**
Create a subtle abstract background shape set for a UI loading state. Use soft rounded blobs and gentle gradients in warm bright colors (tangerine, lemon, mint). No text, no numbers, no photo realism. Keep shapes minimal and airy. Output as a transparent PNG, 1600x900.

## 6) Small Icon Set (Optional)
**Placement:** Sidebar nav icons or feature cards in `/app`.
**Asset location:** `public/assets/icons/` (save each icon as `bowl.webp`, `calendar.webp`, `grocery-bag.webp`, `checklist.webp`, `timer.webp`, `recipe-card.webp`)
**Purpose:** Consistent visual language.
**Prompt:**
Create a cohesive set of 6 minimal flat icons for a meal prep planner: bowl, calendar, grocery bag, checklist, timer, and recipe card. Use consistent stroke weight or solid shapes, rounded corners, bright palette (tangerine, mint, sunny yellow), and soft shadows. No text, no numbers, no photo realism. Output as individual 512x512 PNGs on transparent backgrounds.
