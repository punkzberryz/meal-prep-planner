# Next.js Quality Checks

## Overview

Use this skill whenever code changes are made in a Next.js codebase and you need to run standard quality checks.

## Workflow

1. Format the code
   - Run `pnpm format`.

2. Lint the code
   - Run `pnpm lint`.
